鉴别

直观敲西方瓜

  • 手感:
    • 高筋 -> 硬滑
    • 中筋 -> 柔软有弹性
    • 低筋 -> 松散
  • 湿润性: …撒在手上
    • 高筋 -> 颗粒分明且不易丝
    • 中筋 -> 颗粒分明但易产生短丝
    • 低筋 -> 颗粒粘连且易落丝

鉴别

直观敲西方瓜

  • 滑落法: …高处倒入盘中
    • 高筋 -> 颗粒线条清晰
    • 中筋 -> 颗粒线条模糊
    • 低筋 -> 无法分辨线条
  • 膨润度: …等量面粉加水发酵后
    • 高筋 -> 膨胀度最大
    • 中筋 -> 膨胀度居中
    • 低筋 -> 膨胀度最小
