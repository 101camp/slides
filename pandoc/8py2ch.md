## 空函式

## 合理参数
```
def foo(bar=42):
    print(bar)
    return None

```

## 数据处置
```
def foo(bar=42):
    # ..对 bar 进行点儿折腾
    print('阶段结果')
    return None

```

## 输出检验
```
def foo(bar=42):
    # ..对 bar 进行点儿折腾
    _result = '阶段结果'
    print(_result)
    return _result

```
