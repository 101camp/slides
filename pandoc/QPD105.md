## 触发
```
企业/WX群/ ... 任何即时场景中
         *
        / \
        |  +- 触发策划/创新/改进/任务/...

```

## 任务细化
```
        |   |  立即转入 git.code.tencent.com/presto
        |   |               
        |    *- => README/issues 进行具体描述/追踪/讨论/...
        |   / \       +- 随时增补记要     
```

## 公开追查
```
        ^   ^   V     +- 定期:周1/3/5 1642 在线表格 校对进展
        |   |   |          `- PrestoDriect-工程追踪 
```

## G.F.M
```
        ^   ^   +- 代码 追踪在各自对应仓库中
        |   |   |     +- 阶段提交 sandbox
        |   |   |     |       `- 触发 Jenkins 自动构建事务...
        |   |   |     |
        ^   ^   V     +- 日常 dev_<UID> 分支中
        |   |   |           +- commit log 前缀对应 Issue 编号
        |   |   |          /
        |   +---+------<--+ (将自动解析为对应 Issue 链接)
```
