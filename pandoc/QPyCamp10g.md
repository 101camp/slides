## 伪代码v2
> 自然陈述


```
生成 我的数
输入 你猜的
对比 (我的数 和 你猜的)
如果 没猜中:
    输出 大小判定
    继续玩
如果 猜中:
    输出 "猜对了"

```

## 伪代码v3
> 自然陈述


```
生成 我的数
输入 你猜的

如果 没猜中:
    如果 你猜的 > 我的数 : 输出 "大了"
    如果 你猜的 < 我的数 : 输出 "小了"
    否则:
        输出 "猜对了"
        退出
    继续玩

```
